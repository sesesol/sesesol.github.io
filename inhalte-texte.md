# راهنمای محتوای وب‌سایت
# Inhalte & Texte - Content Guide

## نکات مهم برای نوشتن محتوا / Wichtige Hinweise

این سند شامل پیشنهادات و راهنمایی‌هایی برای تکمیل محتوای وب‌سایت است.
Dieses Dokument enthält Vorschläge und Anleitungen zum Vervollständigen der Website-Inhalte.

## صفحه اصلی / Startseite

### عنوان اصلی / Hauptüberschrift
```
مراقبت پزشکی با تجربه و اعتماد
Medizinische Versorgung mit Erfahrung und Vertrauen
```

### متن توضیحی / Beschreibungstext
```
ارائه خدمات تخصصی پزشکی با بالاترین استانداردهای کیفی در یزد. سلامت شما اولویت ماست.
```

Alternative Texte:
- "با بیش از X سال تجربه در خدمت سلامت شما"
- "تخصص، تجربه و مراقبت در یک مکان"
- "سلامتی شما، مسئولیت ما"

## درباره پزشک / Über den Arzt

### بخش معرفی / Vorstellung

**نمونه متن:**
```
دکتر [نام] فارغ‌التحصیل دانشگاه علوم پزشکی [نام شهر] است که پس از اتمام دوره تخصص 
خود در [رشته تخصصی]، بیش از [تعداد] سال در زمینه [حوزه تخصصی] فعالیت داشته است.

هدف اصلی من همیشه این بوده که به بیماران کمک کنم تا بهترین نتیجه درمانی را با 
کمترین استرس و نگرانی تجربه کنند. معتقدم که پزشکی موفق تنها به دانش فنی نیاز ندارد، 
بلکه نیاز به گوش دادن، همدلی و احترام به بیمار دارد.
```

### تحصیلات / Ausbildung

**قالب:**
```
- دکترای حرفه‌ای پزشکی (MD): دانشگاه [نام]، سال فارغ‌التحصیلی: [سال]
- دوره تخصص [رشته]: بیمارستان [نام]، [سال شروع - سال پایان]
- فلوشیپ [در صورت وجود]: [نام مرکز]، [سال]
- دوره‌های تکمیلی: [نام دوره‌ها]
```

### تجربه کاری / Berufserfahrung

**نمونه:**
```
۱۴۰۰-کنون: مدیر و متخصص مطب تخصصی در یزد
۱۳۹۵-۱۴۰۰: همکاری با بیمارستان [نام]
۱۳۹۲-۱۳۹۵: دستیار و رزیدنت در بیمارستان [نام]
```

## خدمات / Leistungen

### توضیحات کلی / Allgemeine Beschreibung

**برای هر خدمت:**

**۱. تشخیص / Diagnostik**
```
ما با استفاده از جدیدترین روش‌های تشخیصی و تجهیزات پیشرفته، تشخیص دقیق و 
به موقع بیماری‌ها را تضمین می‌کنیم. این شامل معاینات کامل بالینی، تجویز آزمایشات 
تخصصی و تفسیر نتایج است.
```

**۲. مشاوره / Beratung**
```
در جلسات مشاوره، وقت کافی برای شنیدن نگرانی‌ها و سوالات شما اختصاص می‌یابد. 
ما برنامه درمانی را با توجه به شرایط خاص شما طراحی می‌کنیم و تمام گزینه‌های 
درمانی را به زبان ساده توضیح می‌دهیم.
```

**۳. درمان / Behandlung**
```
درمان‌های ما بر اساس آخرین دستاوردهای علمی و استانداردهای بین‌المللی است. 
ما از روش‌های مؤثر و ایمن استفاده می‌کنیم و در تمام مراحل درمان، شما را همراهی 
می‌کنیم.
```

**۴. پیگیری / Nachsorge**
```
مراقبت از شما پس از اتمام درمان ادامه دارد. با ویزیت‌های منظم و پیگیری وضعیت 
سلامتی، اطمینان می‌یابیم که روند بهبودی به خوبی پیش می‌رود.
```

### بیماری‌های تحت درمان / Behandlungsschwerpunkte

**نمونه برای یک متخصص داخلی:**
```
- دیابت و اختلالات متابولیک
- فشار خون و بیماری‌های قلبی-عروقی
- اختلالات تیروئید
- بیماری‌های گوارشی
- آرتریت و بیماری‌های روماتولوژیک
- آسم و بیماری‌های تنفسی
```

**برای سایر تخصص‌ها، لیست را مطابق حوزه کاری خود تنظیم کنید.**

## مطب / Praxis

### راهنمای دسترسی / Anfahrtsbeschreibung

**قالب:**
```
از میدان [نام میدان] به سمت [جهت] حرکت کنید.
پس از گذشتن از [نشانه مشخص مثل بانک، داروخانه، پارک]، وارد [نام خیابان] شوید.
ساختمان ما در سمت [راست/چپ] با پلاک [شماره] قرار دارد.
ورودی از [طرف/خیابان] است.
```

### پارکینگ / Parkmöglichkeiten

**نمونه‌ها:**
```
- پارکینگ اختصاصی با ظرفیت X خودرو در حیاط ساختمان
- پارکینگ عمومی در فاصله Y متری
- امکان پارک موقت در خیابان [نام]
- نزدیک‌ترین پارکینگ عمومی: [آدرس]
```

## تماس / Kontakt

### پیام‌های خودکار / Automatische Antworten

**برای فرم تماس (پیام موفقیت):**
```
پیام شما با موفقیت ارسال شد.
به زودی با شما تماس خواهیم گرفت.
از صبر و شکیبایی شما سپاسگزاریم.
```

**برای فرم تماس (پیام خطا):**
```
متأسفانه خطایی رخ داد.
لطفاً دوباره تلاش کنید یا با ما تلفنی تماس بگیرید.
```

## نکات مهم برای نوشتن محتوا / Wichtige Content-Tipps

### ۱. زبان ساده و قابل فهم / Einfache Sprache
- از اصطلاحات پزشکی پیچیده پرهیز کنید
- هر اصطلاح تخصصی را توضیح دهید
- جملات کوتاه و واضح استفاده کنید

### ۲. تمرکز بر بیمار / Patientenfokus
- به جای "ما خدمات ارائه می‌دهیم"
- بنویسید "شما دریافت خواهید کرد"

### ۳. اعتمادسازی / Vertrauensaufbau
- تجربیات مثبت را برجسته کنید
- شفاف و صادق باشید
- تعهدات خود را روشن کنید

### ۴. فراخوان به اقدام / Call-to-Action
**نمونه‌های خوب:**
- "همین حالا تماس بگیرید"
- "برای نوبت‌گیری کلیک کنید"
- "سوالی دارید؟ با ما در تماس باشید"

### ۵. SEO (بهینه‌سازی موتور جستجو)
**کلمات کلیدی مهم:**
- نام شهر: یزد
- تخصص: [نام تخصص]
- خدمات: درمان، مشاوره، تشخیص
- بیماری‌های خاص

**نمونه عنوان SEO:**
```
دکتر [نام] - متخصص [رشته] در یزد | مشاوره و درمان [بیماری‌ها]
```

## لیست بررسی محتوا / Content-Checkliste

### قبل از انتشار / Vor Veröffentlichung

- [ ] تمام نام‌ها و عناوین صحیح هستند
- [ ] شماره تلفن و ایمیل درست است
- [ ] آدرس کامل و دقیق است
- [ ] ساعات کاری به‌روز است
- [ ] املا و دستور زبان بررسی شده
- [ ] لینک‌ها کار می‌کنند
- [ ] اطلاعات تخصصی دقیق است
- [ ] هیچ ادعای غیرواقعی وجود ندارد

### به‌روزرسانی منظم / Regelmäßige Updates

**ماهانه:**
- بررسی اطلاعات تماس
- به‌روزرسانی ساعات کاری (در صورت تغییر)

**سالانه:**
- به‌روزرسانی سال copyright
- بازنگری اطلاعات بیوگرافی
- بررسی و به‌روزرسانی لیست خدمات

## نمونه متن‌های کامل / Vollständige Textbeispiele

### متن خوش‌آمدگویی

```
به مطب تخصصی دکتر [نام] خوش آمدید

ما با افتخار خدمات تخصصی [رشته] را در یزد ارائه می‌دهیم. هدف ما فراهم کردن 
بهترین مراقبت‌های پزشکی در محیطی آرام، حرفه‌ای و مبتنی بر احترام است.

با بیش از [تعداد] سال تجربه در زمینه [حوزه تخصصی]، ما به ارائه درمان‌های 
مؤثر، مشاوره‌های دقیق و پیگیری مستمر بیماران خود متعهد هستیم.

سلامت شما برای ما اولویت است.
```

### فلسفه درمانی

```
رویکرد ما در درمان

معتقدیم که هر بیمار منحصر به فرد است و به یک برنامه درمانی شخصی‌سازی شده 
نیاز دارد. به همین دلیل، ما:

• وقت کافی برای شنیدن شما اختصاص می‌دهیم
• تمام گزینه‌های درمانی را به زبان ساده توضیح می‌دهیم
• شما را در تصمیم‌گیری‌های درمانی مشارکت می‌دهیم
• از آخرین دستاوردهای علمی استفاده می‌کنیم
• به حریم خصوصی شما احترام می‌گذاریم

هدف ما این است که نه تنها بیماری شما را درمان کنیم، بلکه به بهبود کیفیت 
زندگی شما نیز کمک کنیم.
```
